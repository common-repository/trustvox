== Description ==

O consumidor precisa confiar para comprar na sua loja. Mas quando você deixa qualquer um avaliá-la, 21% dos reviews são feitos por quem nunca comprou o produto. Por isso, a Trustvox coleta automaticamente somente a opinião de clientes da sua loja, certificando os reviews com o selo Site Sincero!

Certificamos que as opiniões publicadas são reais e sinceras, de quem realmente comprou o produto.

== Installation ==
= Instalação do plugin: =
- Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.
- Ative o plugin.

= Requerimentos: =
- É necessário possuir uma conta no Trustvox e ter instalado o WooCommerce.

= Configurações do Plugin: =
Com o plugin instalado acesse o admin do WordPress e entre em "Configurações" > "Trustvox".



== Frequently Asked Questions ==

= O TrustVox é gratuito? =
Não. É necessário adquirir um plano. [Comece agora](https://site.trustvox.com.br/contato/)

= Eu preciso ter o plugin WooCommerce instalado? =
Sim.

= Este plugin funciona com outros plugins de Loja Virtual para WordPress? =
Não, somente com o WooCommerce.

= Como funciona o TrustVox? =
Somos um certificador de reviews. Mais confiança, mais vendas! [Saiba mais](https://site.trustvox.com.br/)

= Os comentários da minha loja vão ser desabilitados? =
Sim. O sistema de comentário padrão do WooCommerce será desabilitado. Caso você tenha outro sistema como Facebook Comentários/ Disqus/outros recomendamos desabilitar por alguns motivos:)
Quanto menos, melhor. Tanto para o seu cliente quanto para você :):
Seu cliente entenderá que agora ele tem um canal de comunicação único
Será necessário atender apenas por 1 lugar: Você utilizará apenas o painel da Trustvox para isso

= É possível migrar/importar as opiniões que já tenho em minha loja para o TrustVox? =
Não. Porém iremos coletar automaticamente todos os pedidos com o status "Enviado" dos últimos 90 dias.
Como a Trustvox é uma certificadora de opiniões verdadeiras, precisamos garantir que todas as opiniões são reais.

= Preciso instalar o HTML da Trustvox ? =
Não. A instalação é totalmente plug-and-play. A Trustvox irá aparecer automaticamente no detalhe de todas as suas páginas de produtos.

= As vendas são cadastradas automaticamente? =
Sim! O Plugin envia todos os pedidos marcados como "Concluído" no WooCommerce. A Trustvox enviará o email para coletar opinião 5 dias após a data estimada de entrega.[incluir imagem do painel selecionado concluído]

= E as vendas/pedidos antigas(os)? =
Também ;). Iremos importar automaticamente todos os pedidos dos últimos 90 dias com o status "Concluído" a partir da ativação do TrustVox em sua loja.

= Como posso incluir o Selo Site Sincero em minha loja? =
* Sua loja já possui mais de 30 dias e pelo menos 50 opiniões publicadas na Trustvox?
* Se a resposta foi sim para os dois, você já está apto a instalar o Selo Site Sincero em sua loja.
* Para isso, entre em contato com a Trustvox em suporte@trustvox.com.br para gerar o seu Certificado de Sinceridade.
* Assim que obtiver o código, é possível adicioná-lo na aba "Selo Site Sincero", escolher o modo de exibição e pronto :)

Acesse nossa seção Help no endereço: https://help.trustvox.com.br/
